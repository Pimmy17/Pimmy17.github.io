QA Cinema

Solo Project

Group Project as part of QA Academy.
Created both back-end and front-end from scratch.


As part of the final project of the 12 week QA Academy, I was asked, as part of a team of 4,
to create a working website that resembles a typical cinema. We had 9 days to create it using
technologies that we had learned during the 12 weeks. These included Java, Spring Boot, Maven, JUnit,
Mockito, Git, Jenkins, AWS, MySQL, JavaScript, HTML & CSS.

If you try to access the 'Live' version and it doesn't work, then our access to AWS has been stopped.
